# RAG Chatbot using Hugging Face

A simple PDF chatbot that uses Retrieval-Augmented Generation (RAG) with Hugging Face models, FAISS, and Streamlit.

## Features
- Upload and query PDFs
- Uses `flan-t5-xl` via Hugging Face
- Embeddings + FAISS + Streamlit UI

## Setup
1. Clone the repo
2. Run: `pip install -r requirements.txt`
3. Add your `.env` file with Hugging Face token
4. Run: `streamlit run src/app.py`
